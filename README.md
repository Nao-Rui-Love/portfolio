# Naofumi Shimomura Portfolio

## 概要

Web制作の学習成果として制作したポートフォリオサイトです。

自身のプロフィール、制作したWeb作品、使用できる技術、お問い合わせフォームをまとめています。

## Demo

GitHub Pagesで公開しています。

https://nao-rui-love.github.io/portfolio/

## 使用技術

- HTML5
- CSS3
- JavaScript
- EmailJS

## 主な機能

### Home
- ハンバーガーメニュー
- レスポンシブ対応
- ポートフォリオへの入口ページ

### Profile
- 自己紹介
- 経歴紹介

### Works
- 制作作品の紹介
- 使用技術・制作内容を掲載

### Skill
- 習得技術の紹介

### Contact
- お問い合わせフォーム
- EmailJSによるメール送信機能

## 制作目的

Web制作の技術習得と、制作した作品を紹介するために作成しました。

HTML、CSS、JavaScriptを使用し、デザインから実装まで自分で制作しています。

## 制作環境

- Visual Studio Code
- Git / GitHub
- GitHub Pages
